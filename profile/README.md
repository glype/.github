# Glhat Chat application

A free and open source chat app that aims to be good and fun!

## ✨ Features ✨

* Selfhosted
* Free & open source
* Nice UI
* Customizeable

## ✨ Running ✨

* download [gleam](https://gleam.run/)
* git clone [glhat-app/core](https://github.com/glhat-app/core.git)
* `gleam run` in the folder
